# SmartSociety

A modern society management web app.

## Deploy

### Netlify
1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → New Site from Git → select repo → Deploy

### Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import repo → Deploy

No build step required — this is a static HTML site.
